# Preferencias del proyecto

## App ATLAS
- **Siempre que se trabaje con la app ATLAS, hacerlo sobre el archivo `.jsx`** (`ATLAS_v12.jsx`). Es la fuente oficial del código.
- El archivo `ATLAS.html` es solo una versión empaquetada para abrir/probar en el navegador. No editar su lógica directamente; si hace falta, se regenera a partir del `.jsx`.
- Cuando el usuario pida cambios en ATLAS, editar el `.jsx`. Solo tocar el `.html` si lo pide expresamente.
